# データベースディレクトリ

このディレクトリにはNullTaskerのSQLiteデータベースファイルが格納されます。

## ファイル構成

- `database.js` - データベース操作を管理するクラス
- `schema.sql` - データベーススキーマ定義
- `nulltasker.db` - メインデータベースファイル（自動生成）
- `backups/` - データベースバックアップ用ディレクトリ

## 注意事項

- `.db`, `.db-shm`, `.db-wal`ファイルはGit追跡されません
- 初回起動時に`nulltasker.db`が自動的に作成されます
- バックアップは自動的に`backups/`ディレクトリに保存されます
