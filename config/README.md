# 設定ディレクトリ

このディレクトリにはNullTaskerのランタイム設定ファイルが格納されます。

## ファイル構成

- `backups/` - 設定ファイルのバックアップ用ディレクトリ

## ランタイム生成ファイル（Git追跡外）

以下のファイルはアプリケーション実行時に自動生成されます：

- `settings.json` - アプリケーション設定
- `tickets.json` - チケット/タスクデータ（SQLite移行前の旧形式）
- `users.json` - ユーザーデータ（SQLite移行前の旧形式）
- `projects.json` - プロジェクトデータ（SQLite移行前の旧形式）

## 注意事項

- これらのファイルはGit追跡されません
- バックアップファイルも自動的に除外されます
- 現在はSQLiteデータベースに移行済みです
